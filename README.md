# Task Manager

Комплексное приложение для управления задачами на Flutter (Dart) для Android и iOS.

## Возможности

- Создание, редактирование и удаление задач
- Категории задач (Работа, Личное, Учеба, Здоровье, Покупки, Другое)
- Приоритеты задач (Низкий, Средний, Высокий)
- Дедлайны с уведомлениями о просроченных задачах
- Фильтрация задач (Все, Сегодня, Предстоящие, Выполненные)
- Поиск задач по названию и описанию
- Статистика с графиками (круговая диаграмма выполнения, столбчатая диаграмма по категориям)
- Темная и светлая тема
- Локальное хранение данных через Hive
- Swipe-действия для редактирования и удаления

## Требования

- Flutter 3.16+
- Dart 3.0+
- Android SDK / Xcode (для iOS)

## Запуск

```bash
# Установка зависимостей
flutter pub get

# Запуск на Android
flutter run

# Запуск на iOS (требуется macOS + Xcode)
flutter run -d ios
```

## Структура проекта

```
lib/
  main.dart              - Точка входа
  models/                - Модели данных (Task, Category)
  providers/             - State management (Provider)
  services/              - Hive сервис для хранения данных
  screens/               - Экраны приложения
  widgets/               - Переиспользуемые виджеты
```

## Зависимости

- hive, hive_flutter - локальное хранилище
- provider - управление состоянием
- fl_chart - графики и диаграммы
- flutter_slidable - свайп действия
- intl - форматирование дат
- google_fonts - шрифты
